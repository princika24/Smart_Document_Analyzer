# Smart_Document_Analyzer

The Smart Document Analyzer is an AI-powered web application built with Streamlit that allows users to upload documents and instantly gain insights from them.

It can:
✅ Answer questions based on the document’s content
✅ Summarize the document
✅ Extract keywords from the document
✅ Provide a clean, interactive Streamlit UI for seamless user experience

**Features**
1. Document Upload
Supports common formats such as:
* PDF
* TXT
* DOCX

2. Question Answering
Users can ask any question, and the app returns AI-generated answers derived specifically from the uploaded document.

3. Document Summarization
Generates concise summaries to help users quickly understand the document's main ideas.

4. Keyword Extraction
Extracts important:
* Keywords
* Topics
Useful for indexing, metadata creation, and quick comprehension.

5. Streamlit UI
* Clean and intuitive
* Real-time interactions
* Instant responses from the backend model
